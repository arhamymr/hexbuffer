Powerful Features

Everything you need to intercept, inspect, modify, and debug HTTP/HTTPS traffic.

🔒 MITM Proxy

Full man-in-the-middle proxy with automatic CA certificate generation and OS-level trust store integration.

  • ✓ Automatic CA certificate generation with SHA-256 fingerprinting
  • ✓ macOS Keychain and Windows certificate store integration
  • ✓ rustls-based TLS termination for maximum performance
  • ✓ Support for HTTP/1.1 and HTTP/2 protocols

🔍 Traffic Inspection

View every request and response in multiple formats with syntax highlighting and full search capabilities.

  • ✓ JSON viewer with collapsible nodes and search
  • ✓ XML viewer with syntax highlighting
  • ✓ Hex dump viewer for binary content
  • ✓ Image viewer with zoom support

✏️ Request Editing & Repeater

Modify requests and send them again with the powerful Repeater tool. Build complex test scenarios.

  • ✓ Edit headers, query params, and body
  • ✓ Support for multipart form uploads
  • ✓ Request history with unlimited replay
  • ✓ Import from cURL commands

🏷️ Traffic Filtering & Tagging

Find what you're looking for with powerful filters and automatic traffic tagging.

  • ✓ Filter by URL, method, status code, and content type
  • ✓ Client-based filtering for multi-user environments
  • ✓ Custom tagging rules with regex support
  • ✓ Sync and async rule evaluation

⏸️ Breakpoints

Pause requests mid-flow to inspect or modify them before they continue to their destination.

  • ✓ Request and response breakpoints
  • ✓ Modify headers, body, and status codes
  • ✓ Conditional breakpoints based on rules
  • ✓ Breakpoint history and management

📊 Session Management

Save, load, and export traffic sessions for collaboration and archived analysis.

  • ✓ Export to HAR format for browser dev tools
  • ✓ Export to CSV for spreadsheet analysis
  • ✓ SQLite database export for programmatic access
  • ✓ Session compression with ZSTD

🔧 JavaScript Scripting

Write custom scripts to automate tasks and extend AppRecon's functionality.

  • ✓ Boa JavaScript engine for fast execution
  • ✓ Request/response interception hooks
  • ✓ Custom authentication flows
  • ✓ Automated security testing

🤖 AI Integration (MCP)

Connect to Large Language Models via the Model Context Protocol for intelligent traffic analysis.

  • ✓ Real-time traffic analysis
  • ✓ Security vulnerability suggestions
  • ✓ API documentation generation
  • ✓ Custom AI tool development

🌐 Port Scanner

Scan targets for open ports and grab banners to identify services.

  • ✓ Concurrent port scanning
  • ✓ Service banner grabbing
  • ✓ Target management and grouping
  • ✓ Results exportable to traffic history

📝 Brute Force Testing

Test authentication endpoints with multiple attack modes and payload processing.

  • ✓ Sniper, Battering Ram, Pitchfork, Cluster Bomb modes
  • ✓ Payload lists and runtime file support
  • ✓ Number range and character enumeration
  • ✓ URL encoding, Base64, and hash transformations
