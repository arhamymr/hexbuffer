Everything You Need to Debug Traffic

From basic requests to complex security testing, AppRecon has you covered.

🔒

MITM Proxy

Decrypt HTTPS traffic with auto-generated CA certificates. Full control over TLS termination with rustls.

🔍

Traffic Inspection

View requests in multiple formats: JSON, XML, Hex, Images, Video, and GraphQL. Syntax highlighted and searchable.

✏️

Request Editing

Modify headers, body, and parameters on the fly. Send requests again with your changes using the Repeater.

🏷️

Auto Tagging

Create rules to automatically tag traffic. Sync or async evaluation for fast filtering of important requests.

📊

Session Management

Save and load traffic sessions. Export to HAR, CSV, or SQLite for collaboration and archived analysis.

🤖

AI Integration

MCP server support for LLM integration. Analyze traffic patterns and get intelligent insights.

Breakpoints & Scripting

Pause requests mid-flow to inspect or modify them before they reach the destination. Write JavaScript scripts to automate repetitive tasks using the Boa engine.
