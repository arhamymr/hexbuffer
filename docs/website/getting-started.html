Get Started

Install AppRecon and start inspecting traffic in minutes.

Prerequisites

Before installing AppRecon, make sure you have the following:

  • Node.js 18+ For running the development server
  • Rust 1.70+ For building the Tauri backend
  • pnpm Package manager (install via npm i -g pnpm)
  • macOS / Windows / Linux Desktop platform

Quick Install

# Clone the repository
git clone https://github.com/your-repo/apprecon.git
cd apprecon

# Install dependencies
pnpm install

# Run the app
pnpm tauri

4 Simple Steps

Get up and running in minutes

1

Install Dependencies

Run pnpm install to install all dependencies.

2

Configure Certificate

On first launch, add the generated CA certificate to your system's trust store.

3

Set Proxy Address

Configure your browser to use localhost:8080 as the proxy.

4

Start Capturing

Browse the web and watch traffic appear in AppRecon's history view.

Available Commands

Command Purpose
pnpm dev Start Vite dev server on port 1420
pnpm dev:clean Free port 1420 and restart dev server
pnpm build Build Vite frontend for production
pnpm preview Preview built frontend locally
pnpm tauri Run Tauri desktop application
cargo test Run proxy tests sequentially
