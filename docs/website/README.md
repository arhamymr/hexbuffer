# AppRecon - Web Landing Page

## Overview

This folder contains the marketing website for AppRecon. The site is designed to showcase the application's features, provide documentation, and attract users.

## Files

- `index.html` - Landing page with hero section, features overview, and call-to-action
- `features.html` - Detailed feature showcase with technical capabilities
- `getting-started.html` - Installation and quick start guide
- `styles.css` - Shared styling for all pages

## Deployment

The website is a static HTML site that can be deployed to any static hosting platform (GitHub Pages, Netlify, Vercel, etc.).

To preview locally, you can use any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

## Customization

Update the meta tags, links, and content in each HTML file to match your deployment URL and branding preferences.
